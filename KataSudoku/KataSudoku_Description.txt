Kata Sudoku
-----------

Sudoku = Suji wa dokushin ni kagiru "Isolate the number", "Number Place"
The rules for this 9x9 number placing game are:
  - Use numbers 1 - 9
  - Each number exists only once in a row
  - Each number exists only once in a column
  - Each number exists only once in a 3x3 block
  
1. Write a solver method for 9x9 Sudoku puzzle. Given a starting situation (partially solved puzzle) are there multiple solutions?

2. Write a 9x9 Sudoku puzzle generator. Use the solver method to verify the puzzle can be solved. 

3. Have the puzzle generator create difficulties of the puzzle.
